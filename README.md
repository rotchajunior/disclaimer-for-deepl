# Traduzido com DeepL (Checkbox + Aviso)

Este plugin para WordPress permite adicionar facilmente um aviso de tradução em seus posts, páginas e Custom Post Types (CPTs) públicos. Ele é ideal para sites que utilizam tradução automática (como DeepL) e desejam informar isso aos leitores de forma elegante e consistente.

## 🚀 Funcionalidades

-   **Checkbox de Ativação**: Adiciona uma caixa de seleção "Traduzido via plugin?" na barra lateral de edição de posts.
-   **Aviso Personalizável**: Configure o texto que será exibido usando o editor clássico do WordPress (WYSIWYG), permitindo negrito, links e outras formatações.
-   **Posicionamento Flexível**: Escolha se o aviso deve aparecer no **início** ou no **final** do conteúdo do post.
-   **Design Moderno**: O aviso é injetado como um bloco de grupo nativo do WordPress, garantindo compatibilidade com temas de blocos e um design limpo com bordas e cores suaves.
-   **Suporte a CPTs**: Funciona automaticamente em posts, páginas e qualquer tipo de post personalizado que seja público.

## 🛠️ Instalação

1.  Envie a pasta `disclaimer-for-deepl` para o diretório `/wp-content/plugins/`.
2.  Ative o plugin através do menu 'Plugins' no painel administrativo do WordPress.

## ⚙️ Configuração

Para configurar o texto e a posição do aviso:

1.  Acesse **Configurações > Traduzido via plugin** no menu lateral do WordPress.
2.  No campo **Texto do aviso**, insira a mensagem desejada utilizando o editor visual.
3.  No campo **Posição do aviso**, selecione onde a mensagem deve ser exibida.
4.  Clique em **Salvar alterações**.

## 📖 Como usar

1.  Crie ou edite um post, página ou CPT.
2.  Na barra lateral direita (ou no final da página, dependendo da sua versão do WordPress), localize o quadro **Tradução**.
3.  Marque a opção **Traduzido via plugin?**.
4.  Publique ou atualize o post.
5.  O aviso configurado aparecerá automaticamente no local escolhido.
